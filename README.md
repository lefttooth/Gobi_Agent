# 先验知识分类系统

本项目基于 langchain、langgraph、RAG 和 tools，旨在实现一个可扩展的带有先验知识的分类系统。

## 项目结构
- `main.py`：主入口文件
- `rag/`：RAG 相关模块
- `tools/`：工具模块
- `knowledge/`：先验知识管理模块
- `requirements.txt`：依赖列表

## 初始化说明
当前仅搭建项目基础架构和依赖，未实现具体功能。

## 依赖安装
```bash
pip install -r requirements.txt
```
